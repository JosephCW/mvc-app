## Install Dependencies and Start App

Open PowerShell here as admin and run: `npm install`.

Start app with nodemon: `nodemon app.js`.

## Open a Client

1. Open a web browser at: <http://127.0.0.1:3004/> or <http://localhost:3004/>.
